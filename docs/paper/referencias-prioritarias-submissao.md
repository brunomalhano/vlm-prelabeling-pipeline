## Drop-First Policy

If count exceeds 16, remove in this order:

1. Domain-specific surveys not used in discussion or limitations.
2. Secondary architecture citations that duplicate claims already covered by core model papers.
3. Generic annotation-efficiency references with no direct tie to instance pre-labeling.

## Final Freeze Checklist

- Every retained citation supports at least one explicit sentence claim.
- No citation appears only as background decoration.
- Introduction + Related Work + Discussion remain consistent with this same set.
- Bibliography metadata (authors/venue/year/DOI) is complete before submission export.
