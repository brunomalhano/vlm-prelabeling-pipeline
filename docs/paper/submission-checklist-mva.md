# Machine Vision and Applications (Springer, Journal 138) Submission Checklist

Source page: https://link.springer.com/journal/138/submission-guidelines

## Required format and structure

- [ ] Title page includes:
  - [ ] Concise title
  - [ ] All author names
  - [ ] All affiliations (institution, department if applicable, city, state if applicable, country)
  - [ ] Corresponding author with active email
  - [ ] ORCID for authors when available
- [ ] Abstract length is 150-250 words
- [ ] Keywords count is 4-6
- [ ] Main text uses decimal headings with at most 3 levels
- [ ] Abbreviations defined on first use and used consistently

## File package for submission

- [ ] LaTeX source included at submission
- [ ] Compiled PDF included at submission
- [ ] All editable source files included in each revision
- [ ] If using LaTeX template, use Springer Nature template with [iicol] option

## Citation and references style

- [ ] In-text citations are numeric in square brackets
- [ ] Reference list entries are numbered consecutively
- [ ] Reference list includes only cited and published/accepted works
- [ ] DOIs provided as full DOI links whenever available

## Tables and figures

- [ ] Tables numbered with Arabic numerals and cited in order
- [ ] Table captions provided for all tables
- [ ] Figure files named with Fig + number (example: Fig1.eps)
- [ ] Figures cited in order and sized to journal column constraints
- [ ] Vector graphics: EPS preferred; halftones: TIFF preferred; Office formats acceptable
- [ ] Figure labels and lines remain legible at final size

## Supplementary information (if applicable)

- [ ] Supplementary files in standard formats
- [ ] Each supplementary file includes article metadata
- [ ] Mention each supplementary item in main text as Online Resource X
- [ ] Supplementary files have concise captions
- [ ] Large files are compressed as needed (.zip/.gz)

## Declarations and ethics (mandatory where applicable)

- [ ] Include a dedicated Declarations section before references
- [ ] Funding statement included (or explicit no-funding statement)
- [ ] Competing interests statement included
- [ ] Ethics approval / informed consent / animal welfare statements included when relevant
- [ ] Author contributions statement included
- [ ] Data availability statement included (mandatory for original research)
- [ ] Material and/or code availability statement included when relevant

## Policy and integrity compliance

- [ ] Manuscript is original and not under simultaneous consideration elsewhere
- [ ] Reuse of prior material is transparently disclosed
- [ ] All listed authors approved submission and author order
- [ ] Corresponding author responsibilities confirmed
- [ ] AI usage disclosure in Methods when generative AI is used for content generation
